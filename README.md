# Winter Map

![Winter Map]("/images/1_1.webp")

## Description
This mod replaces world map in NFSW with modded, like it's snowy.\
Key Features:
  + Has snow overlay.

### Installation
1. Download the mod files.
2. Extract "Media" folder to ...\Game Files.
3. Enjoy the mod!\
**Note 1: Make sure to back up your original map before installing mod.**
